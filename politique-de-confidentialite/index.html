2. Données collectées

Via le formulaire de contact

Lorsque vous remplissez le formulaire de contact ou réservez un appel, nous collectons :

  • Prénom et nom
  • Adresse email
  • Numéro de téléphone (optionnel)
  • Message libre / description de votre activité
  • URL de votre site web (optionnel)

Finalité : répondre à votre demande et assurer le suivi commercial.
Base légale : intérêt légitime (art. 6.1.f RGPD) + exécution d'un contrat en cours de formation.
Conservation : 3 ans à compter du dernier contact.
Sous-traitant : Go High Level (CRM) — politique de confidentialité GHL. Données hébergées aux États-Unis avec clauses contractuelles types UE.

Via Google Analytics 4 (GA4)

Uniquement si vous avez accepté les cookies d'analyse via la bannière de consentement. GA4 collecte :

  • Pages visitées et durée des visites
  • Source de trafic (Google, direct, réseaux sociaux…)
  • Type d'appareil et pays approximatif
  • Interactions (clics sur les CTAs)

Finalité : mesure d'audience et amélioration du site.
Base légale : consentement (art. 6.1.a RGPD).
Conservation : 14 mois (paramètre GA4 configuré).
Sous-traitant : Google Ireland Ltd — politique Google. L'anonymisation IP est activée.

4. Vos droits

Conformément au RGPD (règlement UE 2016/679), vous disposez des droits suivants :

Accès

Obtenir une copie de vos données personnelles détenues

Rectification

Corriger des données inexactes ou incomplètes

Effacement

Demander la suppression de vos données ("droit à l'oubli")

Portabilité

Recevoir vos données dans un format structuré et lisible

Opposition

Vous opposer à un traitement fondé sur l'intérêt légitime

Limitation

Demander la limitation du traitement dans certains cas

Pour exercer vos droits : michel@troilodigital.be — réponse sous 30 jours maximum.

Si vous estimez que vos droits ne sont pas respectés, vous pouvez introduire une réclamation auprès de l'Autorité de protection des données (APD) belge : dataauthority.be — Rue de la Presse 35, 1000 Bruxelles.

5. Sécurité

Le site est servi exclusivement en HTTPS (TLS 1.3). Les données de contact sont stockées dans Go High Level, plateforme certifiée SOC 2 Type II. Aucune donnée de paiement n'est collectée sur ce site.
